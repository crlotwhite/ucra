Detailed Description

Core API.

Note
Memory Ownership: any pointers returned via UCRA_RenderResult are owned by the engine and remain valid until the next ucra_render() call on the same engine or until ucra_engine_destroy(engine) is called.
Threading: unless stated otherwise by a specific engine, handles are not guaranteed to be thread-safe across concurrent calls.
