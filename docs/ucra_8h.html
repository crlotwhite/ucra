Detailed Description

UCRA Public C API (C99 compatible)

Author
UCRA Development Team
Date
2025

This header defines the core types, error codes, and function signatures for the Universal Choir Rendering API (UCRA).

UCRA is a universal audio synthesis and voicebank rendering library designed for applications such as singing voice synthesis, virtual instruments, and voice conversion tools.

Callback function for pulling PCM data during streaming.

The host application provides this callback to supply note segments and render parameters for the next audio block.

Parameters
